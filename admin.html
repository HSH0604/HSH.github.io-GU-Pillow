發布到網站

📋 發布流程

後台的商品資料儲存在瀏覽器本機，發布時會自動產生完整的前端網頁並推送到 GitHub Pages。

1
在下方輸入 GitHub Personal Access Token（需有 repo 權限）
2
確認 GitHub 帳號與 Repository 名稱正確
3
點擊「🚀 產生並發布」，系統自動產生前端 HTML 並推送
4
等待 1–2 分鐘，GitHub Pages 部署完成後瀏覽器 Ctrl+Shift+R 重新整理
