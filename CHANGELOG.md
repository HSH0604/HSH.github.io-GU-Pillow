# CHANGELOG — 居院抱枕批發目錄

版本號規則：`主版本.次版本.修補版本`
- 主版本：重大架構變更
- 次版本：新增功能
- 修補版本：錯誤修正、小調整

---

## [v1.1.0] — 2026-06-07

### 🐛 錯誤修正

#### 後台管理系統
- **修復編輯功能**：補上遺漏的 `rembg-bar` / `rembg-status` HTML 元素，編輯商品按鈕（✏️）恢復正常
- **修復刪除功能**：同上，刪除按鈕（🗑️）恢復正常
- **新增一鍵去背按鈕**：編輯商品時可直接對現有圖片執行去背

#### 發布系統
- **修復 \[object Promise\] 問題**：`generateHTML()` 改用 `await` 正確等待非同步結果
- **架構改為圖片分離**：圖片存放於 GitHub `images/` 資料夾，`index.html` 從 5MB 縮減至 **67KB**
- **解決 Supabase 資源限制**：原本 7MB base64 超出限制導致發布失敗，改架構後徹底解決
- **Supabase Edge Function 升級 v4**：改用 `encodeBase64`（`@std/encoding`）高效原生編碼
- **新增 `upload_raw` 動作**：瀏覽器送原始位元組，伺服器端編碼，支援圖片與 HTML 上傳
- **智慧圖片快取**：記錄已上傳圖片的雜湊值，重複發布時跳過未變更圖片，大幅加速

#### 前端網站
- **圖片改用 URL**：`<img src="images/{型號}.jpg">` 取代 base64 內嵌，頁面載入更快

## [v1.0.0] — 2026-06-07

### 🎉 首次正式發布

#### 前端網站
- 70 款抱枕商品完整上線
- 5 大系列篩選功能（全部 / 簡約條紋風 / 奢華皮革風 / 拼接撞色風 / 手工工藝風 / 毛絨質感風）
- 商品燈箱放大（點擊卡片展開詳細資訊）
- ESC 鍵關閉燈箱
- 商品 hover 效果（上浮 + 圖片縮放）
- CSS 視覺去背（mix-blend-mode: multiply）
- 響應式設計（支援手機、平板、桌機）
- 字體：Cormorant Garamond + Noto Serif TC + Noto Sans TC

#### 後台管理系統（admin.html）
- **登入驗證**：帳號密碼保護，Session 管理
- **儀表板**：商品總數、系列數、圖片數、缺貨數統計
- **商品管理**：完整 CRUD（新增 / 編輯 / 刪除）
- **欄位**：產品型號 / 類別 / 產品圖片 / 品名 / 商品描述 / 尺寸(CM) / 庫存
- **庫存快速修改**：直接在表格內輸入，即時儲存
- **圖片上傳**：自動壓縮至 800px / JPEG 80%
- **一鍵去背**：瀏覽器端 AI（@imgly/background-removal），完全免費
- **搜尋篩選**：依型號、品名搜尋，依系列篩選
- **匯入 JSON**：含重複料號檢查，有重複則阻擋匯入
- **匯入 Excel**：自動欄位對應，含重複料號檢查
- **匯出 JSON**：完整備份（含 Base64 圖片）
- **一鍵發布**：產生前端 HTML 並透過 GitHub API 推送
- **預覽功能**：發布前先預覽前端網頁
- **部署紀錄**：即時顯示推送狀態
